## ✨ Features

Based on a multi-page HTML structure, the website includes the following dedicated sections:

*   **🏠 Home (`index.html`):** The landing page with a brief introduction.
*   **👤 About (`about.html`):** Detailed background information and my professional journey.
*   **🛠️ Skills (`skills.html`):** A visual representation of my technical abilities and toolset.
*   **💼 Projects (`Projects.html`):** A gallery/showcase of my recent work, filterable using Isotope.
*   **📞 Contact (`contact.html`):** A functional contact form with validation, integrated with Google Maps.
*   **📄 Resume Download:** Direct access to download my latest PDF resume (`DevanshResume.pdf`).

## 🛠️ Tech Stack & Libraries Used

This project is built using vanilla web technologies enhanced by several powerful libraries:

**Core:**
*   HTML5 & CSS3
*   SCSS (Sass Preprocessor for styling)
*   Vanilla JavaScript

**Frameworks & Libraries:**
*   **Bootstrap:** For responsive grid layout and UI components.
*   **jQuery:** For DOM manipulation and plugin support.
*   **Owl Carousel:** For responsive slider/carousel components.
*   **Isotope:** For filtering and sorting project layouts.
*   **Magnific Popup / SimpleLightbox:** For responsive image and media lightboxes.
*   **FontAwesome & Linericons:** For vector icons.

# 🚀 How to Run Locally
Because this is a static website utilizing standard HTML/CSS/JS, it is very easy to run locally.

## 1. Clone the repository:

```
Bash
git clone https://github.com/DSR001915/Personal-Portfolio-Web-Project.git
```

## 2. Navigate to the directory:

```
Bash
cd Personal-Portfolio-Web-Project
```

## 3. Run the project:
* **Method 1:** Simply double-click the index.html file to open it in your default web browser.
* **Method 2 (Recommended):** Use a local server to ensure all assets and scripts load correctly without CORS issues. If using VS Code, install the Live Server extension, right-click index.html, and select "Open with Live Server".
